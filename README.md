# 🍦 Mangiamo Gelato Caffe — Website

Authentic Italian gelato, made fresh daily, a block from the Huntington Beach Pier.
Family-owned since 2009. *Mangiamo — let's eat.*

## Pages
- `index.html` — Home
- `flavors.html` — Flavors & Menu (+ delivery ordering)
- `story.html` — Our Story (the family heritage)
- `catering.html` — Catering (lead-capture quote form + packages)
- `faq.html` — FAQ (gelato education + FAQ schema)
- `visit.html` — Visit / Directions (map, live hours, pier wayfinding)

## Tech
Pure HTML/CSS/JS — no build step. Host anywhere (GitHub Pages, Netlify, Vercel).
- `css/style.css` — design system
- `js/main.js` — nav, scroll reveal, live hours, catering form
- `assets/` — optimized imagery

## Pre-launch checklist
1. Swap placeholder photos for real shop / Nick / gelato shots (keep filenames).
2. Connect the catering form to email/CRM (currently a front-end demo).
3. Confirm the Uber Eats link, hours, and phone number.
